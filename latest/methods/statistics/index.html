    # Base.error — Method.

    error(obs::Observable{T})

    Estimate of the one-sigma error of the observable's mean. Respects correlations between measurements through binning analysis.

    Note that this is not the same as Base.std(timeseries(obs)), not even for uncorrelated measurements.

    See also mean(obs).

    # MonteCarloObservable.iswithinerrorbars — Method.

    iswithinerrorbars(a, b, δ[, print=false])

    Checks whether numbers a and b are equal up to given error δ. Will print x ≈ y + k·δ for print=true.

    Is equivalent to isapprox(a,b,atol=δ,rtol=zero(b)).

    # MonteCarloObservable.jackknife_error — Method.

    jackknife_error(g::Function, obs1, ob2, ...)

    Computes the jackknife one sigma error of g(<obs1>, <obs2>, ...) by performing a "leave-one-out" analysis.

    The function g(x) must take one matrix argument x, whose columns correspond to the time series of the observables, and produce a scalar (point estimate).

    Example:

    g(x) = @views mean(x[:,1])^2 - mean(x[:,2].^2) followed by jackknife_error(g, obs1, obs2). Here x[:,1] is basically timeseries(obs1) and x[:,2] is timeseries(obs2).

    source

Manual

Installation / Updating

To install the package execute the following command in the REPL:

Pkg.clone("https://github.com/crstnbr/MonteCarloObservable.jl")

To obtain the latest version of the package just do Pkg.update() or specifically Pkg.update("MonteCarloObservable").

Example

This is a simple demontration of how to use the package for measuring a floating point observable:

julia> using MonteCarloObservable

julia> obs = Observable(Float64, "myobservable")
Observable "myobservable" of type Float64 with 0 measurements

julia> add!(obs, 1.23) # add measurement

julia> obs
Observable "myobservable" of type  Float64 with 1 measurement

julia> push!(obs, rand(4)) # same as add!

julia> length(obs)
5

julia> timeseries(obs)
5-element Array{Float64,1}:
 1.23
 0.72202
 0.976594
 0.194644
 0.776929

julia> obs[3] # conventional element accessing
0.9765941454987768

julia> obs[end-2:end]
3-element Array{Float64,1}:
 0.976594
 0.194644
 0.776929

julia> add!(obs, rand(995))

julia> mean(obs)
0.5155359040043437

julia> std(obs) # one-sigma error of mean (binning analysis)
0.010119064026297512

julia> saveobs(obs, "myobservable.jld")

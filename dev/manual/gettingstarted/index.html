    Getting started

    Sometimes an example says more than a thousand words. So let's start with one.

    Simple Example

    This is a basic demontration of how to use the package:

    julia> using MonteCarloObservable

julia> obs = Observable(Float64, "myobservable")
Float64 Observable

julia> add!(obs, 1.23) # add measurement

julia> obs
Float64 Observable

julia> push!(obs, rand(4)) # same as add!

julia> length(obs)
5

julia> timeseries(obs)
5-element Array{Float64,1}:
 1.23
 0.7020345826901246
 0.8324330699167801
 0.3443476977005411
 0.6515651169206049

julia> obs[3] # conventional element accessing
0.8324330699167801

julia> obs[end-2:end]
3-element Array{Float64,1}:
 0.8324330699167801
 0.3443476977005411
 0.6515651169206049

julia> add!(obs, rand(995))

julia> mean(obs)
0.498775308615336

julia> error(obs) # one-sigma error of mean (binning analysis)
0.0080702216021926

julia> saveobs(obs, "myobservable.jld")

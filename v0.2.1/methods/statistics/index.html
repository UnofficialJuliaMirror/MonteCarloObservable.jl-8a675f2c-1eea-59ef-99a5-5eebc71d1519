    # MonteCarloObservable.binning_error — Method.

    binning_error(X)

    Estimate of the one-sigma error of the time series's mean. Respects correlations between measurements through binning analysis.

    Note that this is not the same as Base.std(X), not even for uncorrelated measurements.

    For more details, see MonteCarloObservable.Rplateaufinder.

    # MonteCarloObservable.binning_error — Method.

    binning_error(X, binsize)

    Estimate of the one-sigma error of the time series's mean. Respect correlations between measurements through binning analysis, using the given binsize (i.e. assuming independence of bins, Eq. 3.18 basically).
